## How it Works

### 1. Shape Selection
The user selects a shape from the provided interactive menu.

### 2. Input Data
Depending on the selection, the program prompts for dimensions:
* **Circle:** Radius.
* **Rectangle:** Length and Width.
* **Square:** Side Length.

### 3. Precise Calculations
The program uses mathematical formulas to calculate the results accurately.

### 4. Output
Displays the calculated Area and Perimeter clearly to the user.
